# Google-ADB
收录、编写了一些实用的adb小脚本  


注意：使用时都需要下载 platform-tools-latest-windows.zip，并解压到一个文件夹。然后把bat脚本复制到文件夹下，和一大堆的exe在同一目录下。


 *若下载脚本后无法正常运行，请转为ANSI编码。

转换方法:搜索并下载 Notepad++ ，把代码复制粘贴进去。在上方菜单中点击“编码-转为ANSI编码格式”，然后保存文件
